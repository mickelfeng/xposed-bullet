# xposed-bullet

### 基于Xposed框架写的 `子弹短信` 插件

最近子弹短信比较火，但在使用发现非锤子手机有部分功能限制，
于是花些时间开始了下面的研究。  

本插件的主要是开启子弹短信发现附近锤友群的的功能(目前在非锤子手机使用的子弹短信是禁用这个功能的)
